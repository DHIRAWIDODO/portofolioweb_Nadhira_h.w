# Personal Portfolio Website
Website portfolio pribadi yang dikembangkan sebagai bagian dari tugas pembuatan website.  
Perancangan antarmuka dilakukan terlebih dahulu menggunakan Figma sebelum diimplementasikan ke dalam bentuk website statis.

## Desain Figma
Desain antarmuka website ini dapat diakses melalui tautan berikut:  
[https://www.figma.com/file/XXXXXXXX/Personal-Portfolio](https://www.figma.com/design/shwPArepLWXqG07Vu8NEaS/Portofolio?node-id=0-1&t=76k0pLpCC8vFLF3f-1)  

Desain tersebut digunakan sebagai acuan visual dalam proses pengembangan website.

## Teknologi yang Digunakan
- HTML  
- CSS  
- Google Fonts (Plus Jakarta Sans)

## Metodologi Pengembangan
Pengembangan website dilakukan melalui tahapan berikut:
1. Perancangan desain antarmuka menggunakan Figma  
2. Implementasi desain ke dalam kode HTML dan CSS  
3. Hosting dan deployment website agar dapat diakses secara online
