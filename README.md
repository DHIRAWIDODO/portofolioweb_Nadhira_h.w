## Deskripsi Proyek
Website ini merupakan portfolio pribadi yang dikembangkan sebagai bagian dari tugas pembuatan website.

Sebelum proses pengkodean, perancangan antarmuka (UI Design) dibuat terlebih dahulu menggunakan Figma. Desain tersebut kemudian diimplementasikan menjadi website statis menggunakan HTML dan CSS.

Website ini dirancang dengan tampilan yang terstruktur, modern, dan responsif.

## Desain UI (Figma)
Desain antarmuka website dapat diakses melalui tautan berikut:
https://www.figma.com/design/shwPArepLWXqG07Vu8NEaS/Portofolio?node-id=0-1&t=76k0pLpCC8vFLF3f-1

Desain tersebut digunakan sebagai acuan visual dalam proses pengembangan website agar hasil akhir sesuai dengan perancangan awal.

## Teknologi yang Digunakan
- HTML  
- CSS  
- Google Fonts (Plus Jakarta Sans)  

## Metodologi Pengembangan
1. Perancangan desain antarmuka menggunakan Figma  
2. Implementasi desain ke dalam HTML dan CSS  
3. Proses hosting dan deployment menggunakan GitHub Pages
